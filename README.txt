# Project 3 - CS405

This project is not fully functional when accessed directly through the file system. However, it works correctly when served through a local server.

To run the project, open your terminal and navigate to the project directory. Then, start a local server using the following command:
python3 -m http.server

After using the given command, you can use these links for correct results:
http://[::]:8000/project3.html
http://localhost:8000/project3.html

These show incorrect results, (mars is not present):
http://127.0.0.1:5500/project3.html
